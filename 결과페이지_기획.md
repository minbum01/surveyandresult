# 결과 페이지 기획 — 분석 대시보드

## 핵심 방향

"단순 수기 목록"이 아닌 **나만의 합격 진단서** 느낌.
8개 답변 × DB 태그 데이터를 교차 분석해서
숫자와 그래프로 설득력 있는 결과를 보여줌.

---

## 태그 점수 시스템

### 개념
각 답변이 DB 태그에 매핑되고, 태그 보유 수기들의 통계로 점수를 산출.

```
답변 태그 → DB에서 해당 태그 보유 수기 조회
→ 그 수기들의 메타데이터(기간, 시험, 합격여부) 집계
→ 사용자 맞춤 수치 생성
```

### 산출 지표 (하드코딩 → 나중에 DB 연결)

| 지표 | 산출 방법 | 표시 방식 |
|------|-----------|-----------|
| 유사 합격자 수 | Q1+Q4 태그 AND 매칭 수기 수 | "312명의 합격자와 비슷합니다" |
| 평균 합격 기간 | 매칭 수기들의 study_period 평균 | "평균 11.2개월" |
| 초시 합격률 | 매칭 수기 중 초시생 비율 | "67% 초시 합격" |
| 가장 많이 선택한 공부법 | 매칭 수기의 태그 빈도 | 상위 3개 공부법 태그 |
| 과목별 어려움 지수 | 매칭 수기 중 특정 과목 언급 비율 | 5개 과목 바 그래프 |

## 섹션별 상세 기획

---

### [히어로 배너] — 전체 너비, 상단 고정

```
야행성 헌터              NIGHT HUSTLER
나와 비슷한 합격자 312명을 분석했습니다

[평균 준비기간 11.2개월]  [초시 합격률 67%]  [가장 많이 선택: 인강+기출]
```

- 배경: 다크(#111) or 레드(#E50914)
- 숫자 3개를 가로로 나란히 → 임팩트

---

### [A] 진단 점수 — 좌상단

**5개 축 레이더 차트 (SVG로 구현)**

```
        준비도
         ▲
    ╱─────╲
   /  87   \
  │ 시간 ── 베이스
   \  73   /
    ╲─────╱
    멘탈 ── 효율
```

| 축 | 산출 기준 |
|----|-----------|
| 준비도 | 베이스(Q4) 기반 0~100 |
| 시간 | 하루공부시간(Q6) 기반 |
| 멘탈 | 고민(Q8)이 멘탈이면 낮게, 커리큘럼이면 높게 |
| 효율 | 공부방식(Q7) 기반 (기출/회독 = 고효율) |
| 목표일치 | 목표기간(Q5) vs 상황(Q1) 현실성 매칭 |

총점 표시: **"종합 점수 78 / 100"**

---

### [B] 합격자 통계 — 중앙 상단

**나와 같은 조건 합격자 분석**

```
같은 조건 합격자 312명 기준

준비 기간 분포
6개월미만  ████░░░░░░  18%
6개월~1년  ████████░░  41%  ← 최다
1년~1년6개월 ██████░░░░  29%
그 이상    ███░░░░░░░  12%

합격 횟수
초시 합격  ████████    67%
재시 합격  ████        24%
3회 이상   █            9%
```

- 수평 바 그래프 (CSS 애니메이션, width 0→실제값)
- 숫자가 카운트업 되는 효과

---

### [C] 단계별 로드맵 — 우상단

**목표 기간(Q5)에 맞춘 월별 계획**

예시: 6개월~1년 선택 시

```
MONTH 1-2    기초 이론
             국어·영어·한국사 기본서 1회독
             핵심 강사 커리큘럼 시작

MONTH 3-4    기출 집중
             최근 5년 기출 반복
             취약 과목 집중 공략

MONTH 5-6    실전 대비
             동형 모의고사 주 2세트
             오답노트 완성 및 단권화

D-30         마무리
             기출 재회독 + 단권화 반복
             멘탈 관리, 컨디션 유지
```

- 타임라인 세로 형태
- 현재 → 합격 방향으로 시각화

---

### [D] 과목별 난이도 예측 — 좌하단

**내 베이스(Q4) 기반 과목별 예상 체감 난이도**

```
국어   ██████░░░░  60%  (보통)
영어   █████████░  88%  (어려움) ← 노베이스면 영어 높게
한국사 ████░░░░░░  42%  (쉬움)
행정학 ███████░░░  68%  (보통)
행정법 ████████░░  82%  (어려움)
```

- 색상: 60% 미만=초록, 60~80%=노랑, 80%이상=빨강
- "영어에 주의가 필요합니다" 코멘트

### [E] 나와 비슷한 합격수기 — 중앙 하단

가로 스크롤 카드 (현재와 동일하되 강화)

카드에 추가:
- **일치 태그 수 / 전체** 표시
- **준비 기간** 표시
- **조회수** 표시

```
┌────────────────────────┐
│ 2024 지방직 9급         │
│ 직장 다니며 6개월 합격  │
│                         │
│ #직장병행 #노베이스     │
│ #인강병행 #멘탈관리     │
│                         │
│ 태그일치 4/5  조회 12.4K│
│ 준비기간: 6개월         │
│              전문보기 → │
└────────────────────────┘
```

### [F] 강사 추천 + 위험 요소 — 우하단

**상단: 강사 TOP 3**

매칭 수기에서 가장 많이 언급된 강사

```
① 신민숙 (국어)   312명 중 198명 선택  ████████░  63%
② 비비안  (영어)   312명 중 174명 선택  ███████░░  56%
③ 이중석 (한국사)  312명 중 159명 선택  ██████░░░  51%
```

**하단: 위험 요소 & 대응**

Q1(상황) + Q8(고민) 기반 자동 생성

```
⚠️ 직장병행 수험생의 주요 실패 요인
   → 야근/회식으로 루틴 붕괴
   대응: 최소 루틴(1시간) 설정 후 지키기

⚠️ 멘탈 관리 주의
   → 비슷한 상황 수험생의 38%가 슬럼프 경험
   대응: 오픈채팅·스터디로 커뮤니티 형성
```

## 하드코딩 → DB 연결 전환 계획

현재는 전부 하드코딩.
백엔드 완성 후 각 섹션의 숫자를 API로 교체:

```
GET /api/analyze?tags=직장병행,노베이스,인강병행,6개월~1년,멘탈관리
→ {
    matchCount: 312,
    avgPeriod: "11.2개월",
    firstPassRate: 67,
    periodDist: {...},
    subjectDifficulty: {...},
    topInstructors: [...],
    topNotes: [...]
  }
```
