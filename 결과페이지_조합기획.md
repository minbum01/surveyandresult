# 결과 페이지 — 조합 기반 개인화 기획

## 핵심 원칙

각 섹션이 **최소 2개 이상의 답변 조합**으로 결과가 달라져야 함.
사용자가 "아 이게 나만을 위한 분석이구나" 느끼려면
같은 상황이라도 베이스/기간/시간이 다르면 다른 결과가 나와야 함.

---

## 섹션별 개인화 설계

---

### 1. 레포트 카드 — 분석 텍스트 (5줄)

**조합 기준: Q1 × Q4 × Q5 × Q7 × Q8 (5개 전부)**

현재는 단순 템플릿. 조합별로 다른 뉘앙스가 나와야 함.

핵심 케이스:

| Q1 상황 | Q4 베이스 | → 메시지 방향 |
|---------|----------|--------------|
| 전업수험생 | 노베이스 | "시간은 충분하지만 기초 구축에 집중해야" |
| 전업수험생 | 전공자 | "유리한 출발점, 단기 합격 충분히 가능" |
| 직장병행 | 노베이스 | "가장 어려운 케이스. 현실적 기간 설정 필수" |
| 직장병행 | 전공자 | "전공 덕분에 직장병행도 1년 내 가능" |
| 육아병행 | 노베이스 | "시간 절대 부족. 압축 효율 학습이 생명" |
| 재학수험생 | 노베이스 | "방학 집중 + 학기 중 유지가 핵심 전략" |

---

### 2. 현실성 점수 (신규 섹션)

**조합 기준: Q1 × Q5 × Q6**

"당신의 목표 기간이 상황과 공부 시간을 고려했을 때 현실적인가?"

```
현실성 점수 = f(상황 난이도, 목표기간, 하루 시간)
```

| 상황 난이도 기준점 | 기간 | 시간 | → 현실성 |
|-----------------|------|------|---------|
| 전업 | 6개월 | 10시간+ | ★★★★★ 충분히 가능 |
| 전업 | 6개월 | 5시간미만 | ★★★☆☆ 빡빡함 |
| 직장병행 | 6개월 | 5시간미만 | ★☆☆☆☆ 매우 어려운 목표 |
| 직장병행 | 1년 | 5~8시간 | ★★★★☆ 현실적 |
| 직장병행 | 1년6개월 | 5시간미만 | ★★★☆☆ 가능하지만 타이트 |
| 육아병행 | 6개월 | 어떤 시간이든 | ★★☆☆☆ 매우 도전적 |
| 육아병행 | 1년6개월+ | 5시간미만 | ★★★☆☆ 현실적 목표 |

표시: 별 5개 + 코멘트 한 줄
별이 낮으면 → "목표 기간 재설정을 권장합니다" 경고

---

### 3. 학습 스타일 적합도 분석 (신규 섹션)

**조합 기준: Q7 (방식) × Q1 (상황) × Q8 (고민)**

선택한 공부 방식이 현재 상황에 얼마나 맞는지 판단.

| Q7 방식 | Q1 상황 | → 적합도 + 이유 |
|--------|--------|----------------|
| 인강병행 | 전업수험생 | ✅ 최적 — 커리큘럼 따라가기 좋음 |
| 인강병행 | 직장병행 | ✅ 적합 — 배속 수강으로 시간 절약 가능 |
| 독학위주 | 노베이스 | ⚠️ 위험 — 방향 없이 혼자 가다 지치는 케이스 多 |
| 독학위주 | 전공자 | ✅ 적합 — 이미 방향을 알고 있어 독학 가능 |
| 노량진현강 | 직장병행 | ⚠️ 주의 — 출퇴근 시간과 충돌 리스크 |
| 노량진현강 | 전업수험생 | ✅ 적합 — 구조적 생활 리듬 형성에 도움 |
| 기출반복 | 노베이스 | ⚠️ 시기 중요 — 기초 없이 기출 먼저면 효율 낮음 |
| 기출반복 | 수능베이스+ | ✅ 최적 — 바로 기출 들어가도 됨 |
| 스터디병행 | 멘탈관리 고민 | ✅ 강력 추천 — 멘탈에 스터디가 특효 |
| 스터디병행 | 시간부족 고민 | ⚠️ 주의 — 스터디 시간이 오히려 낭비될 수 있음 |

표시: 적합도 색상 라벨 + 이유 한 줄

---

### 4. 주간 루틴 시뮬레이션 (로드맵 대체)

**조합 기준: Q1 (상황) × Q6 (하루 시간)**

"당신의 상황에서 현실적인 일주일 루틴은?"

24가지 조합 중 주요 케이스:

**[전업수험생 × 10시간이상]**
```
월~금  09:00 이론 인강 → 12:00 점심 → 13:00 기출 풀이
       18:00 오답 분석 → 20:00 암기 마무리
토     모의고사 1세트 → 오답 분석 집중
일     휴식 + 한국사 암기만
```

**[직장병행 × 5~8시간]**
```
월~금  07:00 출근 전 30분 단어 암기
       점심 1시간 기출 풀이
       퇴근 후 2시간 인강 (핵심만)
토     4~5시간 집중 학습
일     2시간 복습 + 1시간 휴식
```

**[육아병행 × 5시간미만]**
```
새벽   05:00~07:00 황금 루틴 (아이 깨기 전)
낮     아이 낮잠 시간 30~60분
저녁   아이 재운 후 1시간
주말   배우자 협조로 3시간 확보
```

**[재학수험생 × 5~8시간]**
```
학기중  수업 후 도서관 3~4시간 / 주말 6시간
방학   하루 8~10시간 스파르타 모드
```

표시: 요일/시간대 타임블록 시각화

---

### 5. 과목별 공략 순서 (난이도 대체/보완)

**조합 기준: Q4 (베이스) × Q3 (직렬)**

단순 난이도가 아닌 "어떤 순서로 공부해야 하는가"

| Q4 베이스 | Q3 직렬 | → 공략 순서 |
|----------|--------|------------|
| 노베이스 | 행정직 | 한국사 → 국어 → 영어 → 행정법 → 행정학 (쉬운 순) |
| 노베이스 | 사회복지직 | 한국사 → 사회복지학 → 국어 → 영어 → 행정법 |
| 수능베이스 | 행정직 | 행정법 → 행정학 (생소과목 먼저) → 기존 과목 유지 |
| 전공자 | 행정직 | 행정학/행정법 빠르게 → 국영한 집중 |
| 이과출신 | 기술직 | 전공과목 → 국어 집중 → 한국사 → 영어 |
| 토익베이스 | 행정직 | 행정법 → 행정학 → 국어 (영어는 마무리만) |

표시: 1→2→3→4→5 우선순위 번호 + 이유

---

### 6. 핵심 리스크 분석 (강화)

**조합 기준: Q1 × Q5 × Q8 (3개 조합)**

현재는 Q1만 봄. 3개 조합이면 훨씬 정확한 리스크 예측 가능.

| 상황 | 목표기간 | 고민 | → 핵심 리스크 |
|-----|--------|------|-------------|
| 직장병행 | 6개월 | 멘탈관리 | "무리한 목표+피로+고립 → 3개월 내 번아웃 위험 높음" |
| 전업수험생 | 1년+ | 슬럼프 | "장기 전업은 6개월 전후 슬럼프가 거의 필연" |
| 육아병행 | 6개월 | 시간부족 | "목표기간 자체가 비현실적. 1년 이상 재설정 권고" |
| 직장병행 | 1년 | 커리큘럼 | "커리 못 잡으면 1년이 공중분해됨. 강사 1명 고정 필수" |
| 전업수험생 | 6개월 | 공부법 | "단기에 공부법 헤매면 치명적. 검증된 강사 커리 우선" |

표시: 리스크 레벨 (높음/보통/낮음) + 구체적 대응 방안

## 최종 결과 페이지 섹션 구성 (안)

```
[레포트 카드 — 좌측 전체]     [현실성 점수]        [합격자 통계]
  - 페르소나 타입               Q1×Q5×Q6             DB연결
  - 핵심 지표 4개               별점 + 코멘트
  - 분석 텍스트 5줄
                               [합격수기 카드]        [강사 추천]
                                DB연결                 DB연결

                               [주간 루틴]            [학습스타일
                                Q1×Q6                  적합도]
                                타임블록 시각화         Q7×Q1×Q8
```

## 경우의 수 커버리지

| 섹션 | 조합 기준 | 경우의 수 |
|------|---------|---------|
| 분석 텍스트 | Q1×Q4×Q5×Q8 | 6×5×5×6 = 900가지 |
| 현실성 점수 | Q1×Q5×Q6 | 6×5×4 = 120가지 |
| 학습스타일 적합도 | Q7×Q1×Q8 | 6×6×6 = 216가지 |
| 주간 루틴 | Q1×Q6 | 6×4 = 24가지 |
| 과목 공략 순서 | Q4×Q3 | 5×7 = 35가지 |
| 핵심 리스크 | Q1×Q5×Q8 | 6×5×6 = 180가지 |

→ 모든 조합을 다 구현하면 사용자마다 완전히 다른 결과 페이지

## 구현 전략

전부 하드코딩 테이블로 관리.
경우의 수가 많아 보이지만 핵심 케이스만 정의하고
나머지는 가장 가까운 케이스로 fallback.

```javascript
// 예시
const REALISM = {
  '전업수험생': {

                  '하루10시간미만': { score:3, msg:'빡빡하지만 불가능하지 않습니다.' } },
    '6개월~1년': { '하루10시간이상': { score:5, msg:'여유있는 목표입니다.' },

    ...
  },
  '직장병행': {
    '6개월미만': { '하루10시간미만': { score:1, msg:'⚠️ 매우 도전적. 기간 재검토를 권장합니다.' } },
    ...
  }
}
```
