# 해커스 공무원 합격수기 크롤링 구조

```
https://gosi.hackers.com/html/mmove.htm?id=exam_passnote_new&m=&cate=&cate2=&cate3=&cate4=&cate5=&cate6=&idx=&user_id=&search_opt=&search_txt=&hb_year=&page={페이지번호}
```

## 총 규모
- 전체 약 106페이지 (확인 완료)
- 페이지당 15개 게시글
- **총 약 1,590개 수기**

---

## 목록 페이지 구조

### page=1 (첫 페이지만 다름)
- 게시글 위치: `tr[10]` ~ `tr[24]`
- 상단 tr[1]~tr[9]는 공지/헤더

### page=2 이후
- 게시글 위치: `tr[2]` ~ `tr[16]`

### 각 게시글에서 추출하는 것
| 항목     | XPath (tr[N] 기준)                                                                                         |
|----------|-------------------------------------------------------------------------------------------------------------|
| 제목+링크 | `//*[@id="wrapper"]/div[4]/div[2]/div[8]/div[1]/div/div/table/tbody/tr[{N}]/td[1]/a`                        |
| 조회수    | `//*[@id="wrapper"]/div[4]/div[2]/div[8]/div[1]/div/div/table/tbody/tr[{N}]/td[3]`                          |

## 개별 게시글 구조

### 요약 정보 (상단 메타데이터)
- XPath: `//*[@id="wrapper"]/div[4]/div[2]/div/div/div[1]/div/div/div[1]`
- 형식: `키 : 값` (줄바꿈 구분)

추출되는 필드 7개:
| 필드       | 예시 값                  |
|-----------|--------------------------|
| 카테고리   | 최종합격수기              |
| 응시시험   | 국가직 / 지방직 / 서울시   |
| 급수       | 7급 / 9급                |
| 응시지역   | 서울 / 전국 / 인천·경기    |
| 응시직렬   | 행정직 / 관세직 / 전산직   |
| 수험기간   | 6개월 이상~1년 미만        |
| 합격년도   | 2025년                   |

### 본문 내용
- XPath: `//*[@id="wrapper"]/div[4]/div[2]/div/div/div[2]`
- 텍스트 전체 추출 (이미지 제외)
