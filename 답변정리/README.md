# 답변 정리 폴더

> 이 폴더는 result.html 의 각 동적 영역(no.NN) 이 어떤 답변(질문)을 기준으로 어떤 결과를 내는지 정리하는 곳입니다.
> 마스터 기획서(`../결과페이지_마스터.md`) 가 큰 그림을, 이 폴더는 영역별 디테일·검증·답변 사전입력을 담당합니다.

## 파일 구조

| 파일 | 용도 |
|---|---|
| `no01_매핑표.md` | (예정) Q1 × Q3 = 42칸 한 줄 요약 카피 |
| `no02_매칭근거.md` | no.02 (3~4줄 후기) 매칭 알고리즘·폴백 단계·실제 결과 |
| `_no02_demo_audit.json` | DEMO 5개에 대한 매칭 시뮬 결과 (스크립트가 자동 생성) |

`_` 로 시작하는 파일은 자동 생성물.

## 작업 규칙

- 각 영역(no.NN) 마다 별도 `.md` 파일 작성
- 사람이 직접 채우는 카피는 일반 `.md`, 자동 생성물은 `_` prefix
- 사용자(이민범)가 채울 빈 칸이 있으면 `_빈칸_` 같이 표시
