# 결과 페이지 — 페르소나 영역 기획

> 작성일: 2026-04-28
> 대상: result.html 좌측(또는 상단) 페르소나 카드 영역
> 목적: 사용자가 1~2초 안에 "이건 진짜 나에 대한 이야기다" + "진짜 합격생들의 기록이 있다"고 느끼게 하기

---

## 0. 절대 원칙

| 금지 | 사용 |
|---|---|
| ❌ N명 / N% / 평균 N개월 같은 정확한 수치 | ⭕ "여러 합격생", "다수의 분들", "절반 이상" |
| ❌ "당신은 ~입니다" 류 입발린 칭찬 | ⭕ 합격생이 직접 쓴 실제 워딩 인용 |
| ❌ "힘들지만 해낼 수 있어요" 일반론 | ⭕ 사용자 답변 조합 기반 구체 묘사 |
| ❌ 어렵다 / 약하다 / 포기 / 위험 | ⭕ 강점·자산·길로 리프레임 |

→ **숫자 빼고 워딩으로 승부**. 신뢰는 통계가 아니라 진짜 합격생 한 줄에서 옵니다.

---

## 1. 페르소나 영역에 들어가면 좋을 것 (후보 목록)

### A. 페르소나 칭호 (Identity Name)
- 입력: Q1
- 형식: 한 줄 한글 칭호 + 한 줄 영문 부제
- 톤: 무게감 있게, 과장 없이, 합격생들이 자기 상황을 부르던 말투에서 채집
- 매핑(예시 — 합격수기 표현 기반):

  | Q1 | 칭호(안) | 영문 부제 |
  |---|---|---|
  | 직장병행 | 퇴근 후 헌터 | After-work Hunter |
  | 전업수험생 | 시간을 진하게 쓰는 사람 | Full-time Runner |
  | 재학수험생 | 캠퍼스 위의 도전자 | Campus Challenger |
  | 육아병행 | 새벽의 사람 | Dawn Maker |
  | 퇴직수험생 | 두 번째 출발선의 사람 | Second-line Starter |
  | 아르바이트병행 | 두 갈래를 동시에 걷는 사람 | Twin-track Runner |

### B. 사용자 답변 태그 칩 (Identity Tags)
- 입력: Q1~Q8 라벨
- 형식: 8개 칩, 정체성 핵심 3개(Q1·Q3·Q4)는 골드 강조
- 의도: "내가 고른 게 그대로 정체성이다" 라는 시각적 인증

### C. 메인 인용구 (Hero Quote) ★ 가장 중요
- 출처: 매칭 점수 최상위 합격수기 1편의 진짜 한 문장
- 추출 기준: 50~140자 / 1인칭 / 일반론 X / 사용자 Q1 일치 우선
- 형식: 큰 따옴표 + 세리프 폰트 + 출처 캡션
- 출처 캡션 예: "— 직장병행, 1년 안 합격수기 중"
- **숫자 한 글자도 들어가지 않는 진짜 워딩이 페이지의 격을 결정**

### D. 페르소나 한 줄 요약
- 입력: Q1 + Q3 또는 Q1 + Q4
- 형식: "[Q1 라벨] [Q3 라벨]을 준비하는 분"
- 예: "직장 다니면서 일반행정직을 준비하는 분"
- 의도: 자기 인식 강화. 부풀리지 않기.

### E. 같은 길을 걸은 합격생 존재감 (수치 없이)
- ❌ "당신과 비슷한 47명" → 절대 X
- ⭕ "같은 길을 걸은 분들의 이야기를 모았습니다"
- ⭕ "여기 들어 있는 모든 문장은 실제 합격수기에서 가져온 것입니다"
- 의도: 데이터가 진짜라는 신호 + 신비감

### F. 가장 자주 함께 적힌 단어 (Top Tag)
- 입력: 매칭 합격수기들의 태그 빈도 1위
- 형식: 한 단어 키워드 + 한 줄 부연
- 예: "이분들이 가장 많이 함께 적은 단어 — #단권화"
       "여러 합격생이 시험 직전 한 권으로 압축한 노트를 가지고 있었습니다."
- 의도: 통계 대신 키워드로 데이터 존재감 전달

### G. 다음 페이지 유도 (Scroll/Turn Hint)
- 형식: 작은 화살표 + 한 줄
- 예: "↓ 그분들이 어떻게 공부했는지, 같은 길의 워딩 그대로"
- 의도: 페이지 1 → 2로 넘어가게

### H. 인쇄용 도장/사인 영역 (선택)
- 형식: 작은 박스 — "발급일 / 다짐 한 줄"
- 의도: 종이로 인쇄됐을 때 소장 가치
- 화면에서는 숨김(`@media print` only)

## 2. 영역 안에 꼭 들어가야 할 것 (필수)

1. **페르소나 칭호 (A)** — 정체성
2. **사용자 답변 태그 칩 (B)** — 자기 확인
3. **메인 인용구 (C)** — 신뢰의 핵심
4. **한 줄 요약 (D)** — 자기 인식
5. **같은 길을 걸은 분들 (E)** — 데이터 존재감 (수치 X)

## 3. 있으면 좋은 것 (옵션)

- 가장 자주 함께 적힌 단어 (F)
- 다음 페이지 유도 (G)
- 인쇄 도장 영역 (H)

## 4. 레이아웃 안 (3안)

### 안 1 — 전통적 좌측 카드 (원본 result.html 계승)
```
┌─────────────────────────┐
│ HACKERS PASS REPORT     │  ← 브랜드
│                         │
│ 퇴근 후 헌터            │  ← 페르소나 칭호 (A)
│ After-work Hunter       │
│                         │
│ 직장 다니면서           │  ← 한 줄 요약 (D)
│ 일반행정직을 준비하는 분 │
│                         │
│ "퇴근하고 학원 가는      │  ← 메인 인용구 (C)
│  게 처음엔 미친 짓       │
│  같았는데, 3개월 지나   │
│  니까 루틴이 됩니다."    │
│   — 직장병행 합격수기 중 │
│                         │
│ #직장병행 #국가직 #행정 │  ← 태그 칩 (B)
│ #공부처음 #1년이내 ...  │
│                         │
│ 같은 길을 걸은 분들의   │  ← 존재감 (E)
│ 진짜 기록을 모았습니다  │
│                         │
│ ↓ 다음 장에서 ...       │  ← 다음 유도 (G)
└─────────────────────────┘
```
- 장점: 인쇄 호환 ↑, 익숙한 양식
- 단점: 인용구가 짧아야 함

### 안 2 — 인용구 풀폭 강조 (편집 톤)
```
┌─────────────────────────────────────────┐
│  HACKERS PASS REPORT             #직장병행│
│                                  #일반행정 │
│  퇴근 후 헌터  ·  직장 다니면서   #공부처음 │
│  일반행정직을 준비하는 분          #1년이내 │
│                                          │
│  ────────────────────────────────────── │
│                                          │
│   "퇴근하고 학원 가는 게 처음엔            │
│    미친 짓 같았는데, 3개월 지나니까        │
│    루틴이 됩니다."                         │
│                                          │
│        — 직장병행 1년 안 합격수기 중       │
│                                          │
│  ────────────────────────────────────── │
│  같은 길을 걸은 분들의 워딩 그대로 ↓        │
└─────────────────────────────────────────┘
```
- 장점: 인용구가 가장 강력하게 와닿음
- 단점: 가로형 페이지 한 면 차지

### 안 3 — 명함형 미니멀
```
┌──────────────────────────┐
│   퇴근 후 헌터              │  ← 큰 칭호
│   직장 다니면서             │
│   일반행정직 준비             │
│                            │
│   "한 줄 인용구..."          │
│   — 출처                    │
│                            │
│   #태그 #태그 #태그          │
└──────────────────────────┘
```
- 장점: 단순, 명함 사이즈로도 가능
- 단점: 정보 밀도 낮음

→ **추천: 안 1 또는 안 2**. 안 1이 인쇄 친화적.

## 5. 실제 인용구 추출 — 채택/제외 기준

채택할 문장:
- 50~140자
- 1인칭 ("저는", "제가") 또는 직접 경험 표현
- 구체 명사 (시간·과목·강사·루틴) 포함
- 사용자 Q1과 같은 상황 합격수기 우선
- 일반론·교과서 표현 X

제외할 문장:
- 인사·감사·자기소개
- 통계·점수·수치 포함
- 추상적 결심 ("열심히 했습니다")

좋은 인용구 예 (직장병행 실제 수기에서):
> "직장 병행, 초시였고 퇴근 후 7시~12시나 새벽 1시까지 공부했고, 주말에는 많이 쉬어가면서 해서 하루 10시간 정도 했던 것 같습니다."

→ 단, **숫자가 들어간 문장은 페르소나 영역에서는 피하고** Page 2 학습 패턴 영역에서 활용. 페르소나 영역은 "감정·태도·정체성"이 담긴 문장 우선.

좋은 페르소나 영역 인용구 예:
> "퇴근하고 학원 가는 게 처음엔 미친 짓 같았는데, 3개월 지나니까 루틴이 됩니다."

> "처음엔 막막했지만, 같은 처지의 합격수기를 읽으면서 길이 보였습니다."
