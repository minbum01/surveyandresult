# 매칭 영역 사전 (Area Dictionary)

`persona.html` / `result.html` 의 동적 영역(매칭 후기 의존)을 식별자·이름·매칭 키로 통합 정리.
**대화 시 이 표의 코드(`no.XX`)와 이름을 그대로 사용한다.**

- 단일 매핑 출처: `후기태그_전체.md` → `TAG_EXPANSION`
- 후보 풀: `reviews_data.json` (passage 보유 2,106건)

---

## 페르소나 페이지 (persona.html)

### no.01 — 동일경로 후기
- **위치**: persona.html, 좌측 다크카드 상단 인용박스 (`data-no="02"`)
- **DOM**: `.passage-block` / `#passage-value`, `#passage-source`
- **매칭 키**: `Q1 × Q3 × Q4` (7단계 폴백)
- **출력**: 합격수기 1건의 `passage` 본문(3~4줄) + 메타 라벨
- **데이터**: `passnotes[].passage` (180~350자 + 격려 키워드 가산)
- **폴백**: Q1×Q3×Q4 → Q1×Q4 → Q1×Q3 → Q3×Q4 → Q1만 → Q4만 → Q3만
- **현황**: 192조합 / 190 매칭 / 2 0건 (육아·공안·토익, 퇴직·공안·토익)

### no.02 — 패턴 인용구
- **위치**: persona.html, 좌측 다크카드 메인 인용구 (`data-no="04"`)
- **DOM**: `.hero-quote-block` / `#hero-quote`, `#hero-quote-source`
- **매칭 키**: `Q5 × Q7(멀티) × Q8` (7단계 폴백)
- **출력**: 합격수기 1건의 `quote` 짧은 인용구 + 메타 라벨
- **데이터**: `passnotes[].quotes[]` (감정·태도 키워드 가산: 루틴/마음/스스로/처음/포기하지 등)
- **폴백**: Q5×Q7×Q8 → Q7×Q8 → Q5×Q7 → Q5×Q8 → Q7만 → Q8만 → Q5만

## 결과 페이지 PAGE 1 — 좌측 다크카드 (result.html)

### no.03 — 공감 통계
- **위치**: result.html PAGE 1 좌측 다크카드
- **DOM**: `.empathy-stat`
- **매칭 키**: `Q1 × Q3 × Q4`
- **출력**: 한 문장 + 강조 수치 (예: "비슷한 합격자의 **73%**가 한 번은 포기하려 했습니다.")
- **데이터**: 매칭 후기 중 멘탈/슬럼프/포기 관련 태그 보유 비율 집계
- **상태**: 미구현 (정적 데모)

### no.04 — 4개 지표
- **위치**: result.html PAGE 1 좌측 다크카드
- **DOM**: `.metrics-grid` (4 cards)
- **매칭 키**: `Q1 × Q3 × Q4`
- **출력**: 유사합격자 수 / 평균 준비기간 / 초시 합격률 / 평균 하루 순공
- **데이터**: 매칭 합격수기 집계 (Q5/Q6/초시 비율)
- **상태**: 미구현 (정적 데모)

### no.05 — 종합 분석문
- **위치**: result.html PAGE 1 좌측 다크카드
- **DOM**: `.analysis-text`
- **매칭 키**: `Q1 × Q3 × Q4` (no.04 지표 풀어쓴 것 + Q8 언급)
- **출력**: 4~5문장 분석 글 ("당신과 비슷한 조건의 합격자 47명 중 58%가...")
- **상태**: 미구현 (정적 데모)

## 결과 페이지 PAGE 1 — 우측 분석 패널 (result.html)

**※ 페이지 의도: 다른 후기를 보고 "나도 할 수 있겠다" 느끼게 하기 — 수치 직접 노출 ❌, 평가/판정 워딩 ❌**

### no.06 — 같은 출발점에서 시작한 분들
- **위치**: page1_right.html 우측 첫 번째 카드
- **DOM**: `.section-card` 첫째 / `.cards-row` (.mini-card × 2)
- **매칭 키**: `Q1 × Q4`
- **출력**: 분위기 좋은 후기 2장 — 제목 + 본문 발췌 120자
- **점수**: 격려/시작 키워드(처음/막막/시작/다행/됐/버텼/꾸준 등) + passage 길이 130~350자

### no.07 — 흔들렸을 때 다잡은 한 마디
- **위치**: page1_right.html 우측 두 번째 카드
- **DOM**: `.section-card` 둘째 / `.quote-block` `#sec07-body`
- **매칭 키**: `Q8` → Q8 주제 키워드 매칭 quote 1건
- **Q8별 키워드**: 멘탈관리=마음/불안/평정/다잡 등 (Q8_QUOTE_KW 사전 사용)
- **출력**: 키워드 칩 + 짧은 인용구(50~130자) + 후기 제목

### no.08 — 첫 한 달 풍경
- **위치**: page1_right.html 우측 세 번째 카드
- **DOM**: `.section-card` 셋째 / `.excerpt-block` `#sec08-body`
- **매칭 키**: `Q1 × Q3` → 본문에서 "첫 달/처음에/처음 시작/초반에" 들어간 문장 추출
- **출력**: 본문 발췌 1문장 (60~150자 우선) + 후기 제목

### no.09 — 포기 직전, 다시 책상 앞
- **위치**: page1_right.html 우측 네 번째 카드
- **DOM**: `.section-card` 넷째 / `.quote-block` `#sec09-body`
- **매칭 키**: `Q1` 기반, 슬럼프 키워드(포기하지/버텼/그래도/다잡/놓지/한계 등) 매칭 quote 1건
- **출력**: 키워드 칩 + 인용구 + 후기 제목

### no.10 — 꾸준함을 말하는 한 줄
- **위치**: page1_right.html 우측 다섯 번째 카드
- **DOM**: `.section-card` 다섯째 / `.quote-block` `#sec10-body`
- **매칭 키**: `Q1~Q8 일치도 5점 이상` 상위 60건에서 꾸준 키워드(꾸준/매일/하루하루/스스로/한 발/조금씩/놓지) 매칭 quote 1건
- **출력**: 키워드 칩 + 인용구 + 후기 제목

## 결과 페이지 PAGE 2 (result.html)

### no.11 — 유사 후기 카드 (3건)
- **위치**: PAGE 2 상단
- **DOM**: `.stories-section` > 3 × `.story-card`
- **매칭 키**: `Q1~Q8 전체` (일치 개수 점수로 정렬, 상위 3건)

### no.12 — 합격자 후회
- **위치**: PAGE 2 하단 그리드 좌
- **DOM**: `.insight-card` (후회)
- **매칭 키**: `Q3` (직렬 기반 후회 통계)
- **출력**: 2개 후회 항목 (X%가 언급 + 인용 + 권장)
- **상태**: 미구현

### no.13 — 합격자 첫 달
- **위치**: PAGE 2 하단 그리드 중앙
- **DOM**: `.insight-card` (첫 달)
- **매칭 키**: `Q1 × Q3` (상황+직렬별 첫달 액션)
- **출력**: 4개 액션 번호 리스트
- **상태**: 미구현

### no.14 — 어려운 과목
- **위치**: PAGE 2 하단 그리드 우
- **DOM**: `.hard-subject`
- **매칭 키**: `Q3` (직렬별 과목 난이도 분포)
- **출력**: 5개 과목 막대 + 팁 1줄
- **상태**: 미구현

### no.15 — CTA 인용구
- **위치**: PAGE 2 하단 CTA Footer
- **DOM**: `.cta-closing-quote`, `.cta-closing-source`
- **매칭 키**: `Q1~Q8` → quote (no.02 와 유사하나 닫는 톤·짧은 길이)
- **출력**: 1~2문장 인용구 + "— X·Y·Z 합격자 수기 중"
- **상태**: 미구현
