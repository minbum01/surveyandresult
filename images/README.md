# 사진 교체 가이드

survey.html 에서 사용하는 모든 사진은 이 `images/` 폴더 안에만 있습니다.
**같은 파일명·같은 확장자(.jpg)** 로 덮어쓰기만 하면 즉시 반영됩니다.

> 다른 확장자(png/webp 등)로 바꾸려면 `survey.html` 안의 해당 경로도
> 함께 수정해야 합니다. 가능하면 `.jpg` 로 통일하는 걸 권장합니다.

## 권장 사양

- 가로형 사진, 가로 800px 이상 (현재 기본값 800x533 내외)
- 형식: JPEG (.jpg) 권장 — 용량 작고 호환성 좋음
- 한 장당 100~250 KB 권장. 너무 크면 로딩 느려집니다.

## 새 옵션을 추가하고 싶을 때

`survey.html` 의 `const Qs = [...]` 배열에서 해당 질문의 `opts:[ ... ]` 에
한 줄을 추가하고, 같은 폴더에 이미지 파일을 넣으면 됩니다.
